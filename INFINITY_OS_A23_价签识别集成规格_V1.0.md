## 二、核心功能 · 5 项 AI 识别能力

```
═══════════════════════════════════════════════════════

[功能 1] 价格数字识别
  · 千问 VL 看价签照片
  · 输出价格数字 (¥ X.X)
  · 准确率目标: 95%+

[功能 2] SKU 关联识别
  · 价签旁通常有产品名称
  · OCR 提取 + 数据库匹配
  · 关联到 INFINITY OS SKU 编码

[功能 3] 异常分级
  · 蓝色提示: 略低于底价 (¥15-17, 底价 ¥17)
  · 黄色警告: 低于成本价 (¥13-15)
  · 红色严重: 极低 (低于 ¥13)

[功能 4] 链路反查
  · 异常价格 → 调一物一码链路
  · 找到这瓶酱油经过的二批商和经销商
  · 自动记录·影响评分

[功能 5] 周期对比
  · 同一门店·同一 SKU·1 周/1 月对比
  · 发现"突然降价"的异常模式
  · 区分: 正常促销 vs 恶意乱价

═══════════════════════════════════════════════════════
```

## 三、技术实现 · 千问 VL Prompt 设计

```
═══════════════════════════════════════════════════════

[Prompt 模板]

你是一个零售货架价签识别 AI。请分析这张价签照片。

输出 JSON 格式:
{
  "products": [
    {
      "sku_name": "六月鲜极鲜 500ml",  // OCR 提取
      "price": 17.50,                   // 当前售价
      "original_price": 19.90,          // 原价 (如有)
      "promotion": "买二送一",           // 促销文字 (如有)
      "confidence": 0.95                // 识别置信度
    }
  ],
  "shelf_position": "B-3 中部黄金位",
  "competitor_nearby": [
    { "brand": "海天", "price": 16.80 }
  ],
  "notes": "价签清晰·灯光正常"
}

═══════════════════════════════════════════════════════

[调用方式]

千问 VL API (qwen-vl-max)
  · 输入: 价签照片 (jpg/png · 最大 10MB)
  · 输出: JSON
  · 延迟: 2-5 秒
  · 成本: ¥0.02-0.05 / 张

每天扫码量预估:
  · 1 万 ICA × 平均 30 次/天 × 1 次价签 = 30 万次/天
  · 月成本: 30 万 × 30 天 × ¥0.03 = ¥270 万/月

═══════════════════════════════════════════════════════

[优化方案]

· 不是每次进店都扫·而是抽查 (每周 1 次/SKU)
· 实际识别量: 30 万 × 30% = 9 万次/天
· 月成本: ¥81 万/月 (可接受)

═══════════════════════════════════════════════════════
```

## 八、风险与应对

```
═══════════════════════════════════════════════════════

[风险 1] 千问 VL 准确率不足 95%
  · 应对: 灰度上线·人工复核·渐进降低复核比例
  · 备选: 切换 GPT-4V 或自训模型 (memory #15 路线 A)

[风险 2] 一线 ICA 不愿扫价签 (多一步操作)
  · 应对: 扫价签触发 +1 任务积分
  · 应对: 简化 UI·一键扫描·秒级返回

[风险 3] 二批商抵触 (被监控)
  · 应对: 透明换返点·正向激励 (V1.4 经济互锁)
  · 应对: 试点 1 区域·验证后扩展

[风险 4] 数据隐私 (千问 VL 上传照片)
  · 应对: 与阿里云签数据保护协议
  · 应对: 自训模型 (长期目标)

═══════════════════════════════════════════════════════
```
