# 新品牌动销 grid 数据格式说明

## 原始 xlsx(给DS看的网格)
- **行** = 品类 × 规格(如 生抽/500ML、红烧/1.8L)
- **列** = 子品牌(六月鲜:经典原酿/轻/遵循自然/纯[=有机])
  - 单一品牌(如味达美)无子品牌:列改为 大品类/中品类 层级
- **每格(一个SKU)** 含:实际业绩 / 1-5月目标 / 6-12月目标 / 达成率 / 标价·实价 / 价差% / 销量

## 解析后 CSV(脚本读取,列名固定)
```
品类, 规格, 子品牌, SKU, 实际, 目标15, 目标15占, 目标H2, 达成, 标价, 实价, 价差, 销量
```
- 实际/目标15/销量 = 数值(脚本会 to_numeric)
- 做了 = 实价非空 且 销量>0
- 达成 = 实际 / 目标15 × 100
- 同一(子品牌×品类×规格)可能有多个SKU → 热力网格按格合并(实际、目标各自求和)

## 规格排序(列:小→大)
脚本 spec_ml():ML照数值;含L(非ML)或KG的 ×1000。据此把规格列从小到大排。

## 品类→颜色映射(白底)
生抽#2563eb 鲜味#e8890c 红烧#dc2626 老抽#7c3aed 蒸鱼#0891b2 有机#16a34a
新品类出现时,在脚本 CAT 字典里加一个鲜明、彼此分得开的颜色即可。
