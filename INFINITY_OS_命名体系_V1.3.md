## 二、ICA 详细命名规范 (V1.3 核心)

### 2.1 ICA 标准命名

| 维度 | 命名 | 使用场景 |
|---|---|---|
| 全称英文 | In-Store Culinary Advisor | 国际化文档 / 海外推广 |
| 缩写英文 | **ICA** | 技术文档 / 系统代号 / 文件名 |
| 标准中文 | 门店美食顾问 | 工牌 / 合同 / 正式文档 |
| 简称中文 | 美顾 | UI 对话 / 一线沟通 |
| 旧名 | 美食顾问 | V1.0-V1.2 使用 · 已升级 |

### 2.2 6 类 ICA 身份矩阵 (memory #16)

```
═══════════════════════════════════════════════════════

2×3 矩阵 = 雇主 × 类型

         欣和雇        经销商雇
常驻      ①           ④
巡回      ②           ⑤
临时      ③           ⑥

英文化:
  全部都叫 ICA · 不细分英文代号
  中文区分: "欣和·常驻" / "经销商·巡回" / 等

ICA app 内部支持身份切换:
  · 6 类身份 ICA 用同一个 app
  · 顶部 dropdown 切换身份
  · 不同身份显示不同任务源/权限

═══════════════════════════════════════════════════════
```

### 2.3 ICA 与其他岗位的边界

```
═══════════════════════════════════════════════════════

岗位类型           缩写     核心职责              与 ICA 关系
────────────────────────────────────────────────────────
门店执行           ICA     烹饪/品牌/AIQL/菜谱     本身
零售业代 (欣和)     SR     销售/陈列/账期           协作 (SFA)
零售业代 (经销商)   DSR    销售/陈列/账期           协作 (SFA)
ICA 督导          ICAS   管 ICA·5星考核·培训        直接上级
区域督导           DOM    管 SR+DSR+ICAS          间接上级
区域经理 (废弃)              并入 DOM                  -
多区域督导         MDOM   跨区域 DOM              二级上级

═══════════════════════════════════════════════════════

边界清晰:
  · ICA = 烹饪+品牌导向 (教/做/评)
  · SR/DSR = 销售导向 (谈/陈列/账期)
  · DOM/MDOM = 管理导向 (督/考/培)

═══════════════════════════════════════════════════════
```

### 2.4 ICA 未来扩展 (Culinary Advisor 系列)

```
═══════════════════════════════════════════════════════

当前 (Phase 1):
  ICA = In-Store Culinary Advisor (零售门店)

未来扩展 (Phase 4-5):
  RCA = Restaurant Culinary Advisor (餐饮 B 端)
        · 服务: 餐厅厨房 / 大厨培训
        · 与第 6 渠道 (餐饮) 配套
  
  HCA = Hotel Culinary Advisor (酒店)
        · 服务: 五星酒店 / 高端宴请
        · 与禾然有机品牌契合
  
  教育市场 / 学校食堂 / 工厂食堂 等:
        · 可继续扩展 _CA 后缀

═══════════════════════════════════════════════════════

设计原则:
  · "_CA" 后缀 = "_Culinary Advisor"
  · 前缀 = 场景代号 (I/R/H/etc.)
  · 全部纳入"Culinary Advisor 角色族"
  · 统一管理 / 统一培训 / 统一品牌资产 (AIQL)

═══════════════════════════════════════════════════════
```
