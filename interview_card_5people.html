02 / QUESTION 1
选 Tech Lead
问题 1 · 选 Tech Lead · 1分钟
▸ 看:独立技术决策能力 + 反思能力
"你过去带过最大的项目是什么?项目里你做了哪些技术决策?这些决策今天回头看·哪个是错的·为什么错?"
▸ 评估什么 Tech Lead必须能独立做技术决策·而不是只听指挥 · 必须能反思过往错误·才能在未来避免重复
✓
能说出具体项目+具体决策(不空话)
✓
能承认"我之前选错了X·因为没考虑Y"
✓
说话有结构·能2-3条说清楚
✗
"我都是按上面要求做的"(没主见)
✗
"没什么大错"(没反思能力)
03 / QUESTION 2
选 Backend / Data
问题 2 · 选 Backend / Data · 1分钟
▸ 看:大数据处理本能
"如果给你SAP的MB51表(几千万行流水数据)·让你算每个经销商每个SKU今天的销售总和·你会怎么写SQL+Java代码?"
▸ 评估什么 Backend/Data必须有大数据处理本能·几千万行不是闹着玩的·不能像处理100行那样写
✓
能说出"GROUP BY 经销商, SKU"+"SUM"
✓
能提到"分批查询·避免内存炸"
✓
能提到"加索引"或"分区"
✗
"我先load到内存里"(不懂大数据)
✗
不知道GROUP BY(基础不行)
04 / QUESTION 3
选 DevOps / WeChat
问题 3 · 选 DevOps / WeChat · 1分钟
▸ 看:稳定性意识 + 用户思维
"如果Agent判断要给经销商发企微推送·凌晨3点系统挂了消息没发出去·你怎么设计才能避免这种情况?"
▸ 评估什么 DevOps必须有7×24稳定性意识 · WeChat必须懂推送可靠性 · 这两个角色都要"防患于未然"
✓
能提到"重试机制·失败再发"
✓
能提到"消息队列·先入库再推送"
✓
能提到"监控告警·挂了自动通知"
✗
"那就重启服务器"(不防患)
✗
"应该不会挂"(没意识)
