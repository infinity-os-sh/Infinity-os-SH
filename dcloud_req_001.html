01 / CONTEXT · Dcloud是什么
和SAP的分工
SAP 负责
• 工厂→经销商(IPO0-4)
• 财务凭证+开票
• 应收账款
• 企业级大账本
Dcloud 负责
• 经销商→二批(IPO5-7)

03 / DIFFERENCE · Dcloud vs SAP
Tech Lead必须知道的3个不同
▸ 差异1 · 接入方式
SAP:数据在Hologres·用SQL查
Dcloud:数据在SaaS平台·用HTTP API查 (每秒限20次·注意限流)
▸ 差异2 · 更新延迟
SAP:T+1 (次日)
Dcloud:准实时 (15分钟延迟) · 因为业务员现场上报
▸ 差异3 · ID体系
SAP:KUNNR客户编号 (10位数字)
Dcloud:distributor_id (UUID格式) · 需要d_distributors表里的sap_kunnr字段做映射 · 这是关键
▸ Tech Lead Dcloud接入的关键动作
① 找IT VP拿Dcloud API Key(和Hologres一起申请)
② 优先接d_stock和d_outflow(最重要2张)
③ 通过d_distributors.sap_kunnr建立SAP↔Dcloud的经销商ID映射表
④ d_outflow里的receiver_region和distributor所在region对比 → 发现窜货
⑤ SAP发货量 vs Dcloud出货量 → 发现压货
