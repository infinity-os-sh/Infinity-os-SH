序言 · 为什么要这部宪法

这几年,公司不是被海天、李锦记打败的,是被自己内部的漏洞打败的:投机者钻空子占便宜 → 诚实、优秀的员工看不惯而出走 → 逆向淘汰 → 企业生命力被自己人一点点耗掉。各器官该负的责任没尽到、有的代偿,身体的生命力就弱了。

我们要的不是"管人",是用 Agent 和机制建一个自动守底线的系统:黑脸由规则来当,不由任何人来当。唯其如此,管理者才能真正商量式、授权式地放手——因为底线有机器在守,无差别、不可求情、不疲倦。

当没人能靠投机得利,全员的能量就只剩一条路:百分之百倾注到对外的市场竞争——这就是"百人一步"。本宪法,为此而立。

两根柱子 · 同等重要 · 缺一根·房子就塌
第一条锚不可动:养生命,非管销售
系统的锚,永远是 SKU 生命力(L1 晶体管:某SKU×店×时的状态)。一切上层——满分、收入、推单、风控——都建在这个锚上。人是 Agent 的传感器,收入是钩子,把 SKU 状态拨强才是目的。锚错,上层全废。
第二条两根柱子,同等重要
绩效柱(养生命)与风险柱(守漏洞)地位相等,不分主从。
2.1
绩效是油门:决定往哪开、开多快——激发人。
2.2
风险是安全带:让你敢踩油门——守底线。安全带绝不能反过来当方向盘(不让惩罚驱动一切),但也绝不是可有可无的次要项(没它,授权就是灾难)。
第三条黑脸由规则当,不由人当
这是本宪法的核心条款。
3.1
守漏洞的判定与处置,由规则/Agent 自动完成:对事不对人、无差别、不可求情、24h不疲倦、透明可申诉。
3.2
没有任何人当黑脸:DS不当、主管不当、HR不自动当。人一旦介入定性,就有人情、求情、选择性执法——漏洞照样在,且授权式管理破功。
3.3
人唯一的位置:边界模糊或申诉时,复核"规则"本身,不复核"人"。规则错就改规则(对所有人生效),绝不为个人开口子。
第四条该管的管死,该放的放开
底线靠机制管死,执行靠授权放开。
4.1
机制焊死漏洞(数据三角互证、可信度门、费用转投),让投机无利可图。
4.2
底线之上,看逻辑对就授权放手,鼓励一线自主把对的事做对。处置永向"诊断+转投",不向"没收+扣薪"——守漏洞≠罚款。
第五条谁定参数:最懂现实、视角够宽者定
规则即参数。定参数的人,必须是最懂该参数背后现实、且视角覆盖其影响范围的人。
谁定什么为什么是他
总部结构+战略规则(阶段标准/红线/反作弊阈值/目标网点分母)影响全局·只有总部看得见全盘战略
OM/DOM辖区目标值+本地校正(带原因·留痕)站在战略与一线接缝·最知本地实情
一线不定参数·只提供原料(盘点/标记/反馈)离现实最近但视角最窄·提供信号不拍参数
第六条指标的正当性,只能是"养生命"
任何指标,必须服务于"把 SKU 状态拨强",而非只服务于"考核/算账/分钱"。一个指标若只为考核分钱、不为养生命,它就是坏指标,迟早逼出动作变形(为达标而刷数)。达成它=做对了这阶段该做的事=SKU 生命力真的上升——指标是"对的事"的量化代言,不是悬在头上的鞭子。
总律 · 生命力恢复链
机制焊死漏洞 → 没人靠投机得利 → 诚实人不再吃亏出走 → 逆向淘汰停止 → 全员能量转向对外竞争(百人一步) → 企业生命力恢复
⚠ 诚实说明 · 当前所处阶段

本宪法是方向与原则的根本大法,已成立、即刻生效。但其完全自动化运行(真Agent主动稽核、主动推送)仍需:真实数据管道全通 + 企微推送 + 学习样本积累。当前为 Layer 1.3+:绩效柱已用真实花雕数据驱动、风险柱机制已成型,但部分仍靠页面/人工过渡。宪法先立,系统按它逐步长齐——方向不变,只是水还在一段段接进来。
