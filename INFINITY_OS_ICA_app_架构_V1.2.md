### 3.2 embed=1 模式 (V2.0 关键发明)

```
═══════════════════════════════════════════════════════

问题: 主壳子 + iframe 嵌入时·内容重复
  · 主壳子顶部已有: Layer 1.3 / 陈晓琳 / Stage / 蝴蝶预警
  · iframe 内 INF-XM/XL/XP 又重复显示

解决: URL 参数 ?embed=1
  · INF-XM/XL/XP 文件加 JS 检测 URL 参数
  · 是 → body 加 .embed-mode 类
  · CSS 隐藏: .layer-badge / .butterfly-bar / .id-mini / 
            .tb-stage / .tb-back / .bottom-fix

效果:
  · 独立打开: 完整显示 (所有诚实标识)
  · 主壳子嵌入: 只显示模块内容·无重复

INF-XD 暂未加 embed (不动现役系统·接受 trade-off)

═══════════════════════════════════════════════════════
```

## 五、与其他 APP 联动机制 (Phase 2 准备)

### 5.1 ICA + SR/DSR 协作

```
═══════════════════════════════════════════════════════

ICA app                        SR/DSR app
─────────────────────────────────────────────
教厨艺 / 做菜谱                  谈进货 / 陈列
品牌资产 (AIQL)                 销售数字 (达成率)
家庭场景渗透                    门店货架管控
红烧肉 = 六月鲜                  六月鲜 SKU 进 8 箱

═══════════════════════════════════════════════════════

联动场景示例:
  · ICA 看到门店货架率低 (来自 INF-XD)
  · ICA 在一本账Tab 发现某 SKU 销量降 (来自 INF-XL r 值)
  · ICA 推送任务给 SR (派单互锁 25 步)
  · SR 接到任务 → 拜访经销商 → 加货
  · 经销商出货 → 门店补货 → 消费者买到
  · 完成 · ICA 拿协作奖金 / SR 拿销售奖金

═══════════════════════════════════════════════════════
```
